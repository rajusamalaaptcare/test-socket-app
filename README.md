How to run this app?

# Server

1) Goto server-app folder
2) install node modules
```
npm install
```

3) To run the application
```
node server.js
```
4) Server runs on port 8080

## Run UI 
1) Goto client-app
2) install node modules
```
npm install
```

3) To run the application
```
npm run dev 
```
4) UI runs on port 5173
5) open http://localhost:5073
